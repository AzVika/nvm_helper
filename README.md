# Node Version Manager (nvm)

List of basic nvm commands

## Основні команди:

Оновлення до останньої версії nvm:
```sh
$ brew install nvm
```

Установка 14 версії node:
```sh
$ nvm install 14
```

Перемкнутися на 14 версію node:
```sh
$ nvm use 14
```

Подивитися поточну версію node:
```sh
$ node --version
```
